HTML
HTML eli HyperText Markup Language, on verkkosivujen rakenteen kuvaamiseen käytetty merkintäkieli.
<html>
<head> <style> p { font-size: 20px; background-color: #e0e0e0; padding: 10px; border-radius: 8px; } </style> </head> <body> <p>Tämä kappale on isommalla fontilla ja harmaassa laatikossa.</p> </body> </html>
Tässä esimerkissä muutetaan kappaleen fonttikoko ja lisätään taustalaatikko.
HEAD
HEAD on HTML-sivun “aivojen” osa: se sisältää kaiken metatiedon, jota selain tarvitsee sivun ymmärtämiseen, mutta jota ei näytetä käyttäjälle.
<head>
<style> h1 { color: red; } </style> </head>
Tämä - muuttaa kaikkien h1-otsikoiden värin punaiseksi.
BODY
tarkoittaa HTML-sivun näkyvää sisältöä — kaikkea sitä, mikä oikeasti ilmestyy ruudulle, kuten teksti, kuvat, linkit ja taulukot. Se on yksi HTML-sivun tärkeimmistä osista, ja jokaisella sivulla saa olla vain yksi body elementti.
<body>
<style> img { width: 200px; border: 2px solid black; } </style> <img src="kuva.jpg" alt="Esimerkkikuva"> </body>
Tässä esimerkissä muokataan kuvan kokoa CSS:llä.
P
HTML:n p-elementti tarkoittaa kappaletta. Selaimet lisäävät automaattisesti tyhjää tilaa ennen ja jälkeen kappaleen, ja elementti toimii lohkotason rakenteena tekstin jäsentämiseen
<body>
<style> p { font-size: 18px; padding: 12px; background-color: #f5f5f5; border-left: 4px solid gray; } </style> <p>Tämä kappale on tyylitelty pehmeällä taustalla ja vasemmalla reunaviivalla.</p> </body>
Tässä esimerkissä muokataan kappaleen sisäistä välistystä ja fonttia, ei väriä.
IMG
HTML:n img-elementtiä käytetään kuvan näyttämiseen verkkosivulla. Se ei “sisällä” kuvaa, vaan luo paikan, johon selain lataa kuvatiedoston annetusta osoitteesta. Tärkeimmät attribuutit ovat src (kuvan polku) ja alt (vaihtoehtoinen teksti).
<body>
<style> img { width: 250px; border-radius: 12px; box-shadow: 0 0 10px rgba(0,0,0,0.3); } </style> <img src="kuva.jpg" alt="Esimerkkikuva"> </body>
Tässä esimerkissä muokataan kuvan kokoa ja lisätään pyöristetyt kulmat.
A
HTML:n A-elementti tarkoittaa ankkuria (anchor) ja sitä käytetään hyperlinkkien luomiseen. Se on yksi tärkeimmistä HTML-tageista, koska sen avulla käyttäjä voi siirtyä sivulta toiselle, samaan sivuun sisäisesti tai avata sähköpostin/puhelun linkistä.
<body> <style> a { display: inline-block; padding: 10px 15px; background-color: #e6e6e6; text-decoration: none; border-radius: 6px; font-weight: bold; } </style> <a href="https://example.com">Tämä on tyylitelty linkki</a> </body>
Tässä esimerkissä muokataan linkin ulkoasua: lisätään taustalaatikko, sisätilaa ja pyöristetyt kulmat.
BR
HTML:n br-elementti tekee yksinkertaisen rivinvaihdon tekstissä — se on tyhjä elementti, jota käytetään silloin, kun halutaan katkaista rivi ilman uutta kappaletta. Tämä on hyödyllistä esimerkiksi osoitteissa, runoissa tai muussa tekstissä, jossa rivijako on olennainen.
<body> <style> br { margin-bottom: 20px; display: block; } </style> Teksti ennen rivinvaihtoa.<br> Teksti rivinvaihdon jälkeen. </body>
Lisätään ylimääräistä väliä rivinvaihdon kohdalle.
H1
HTML:n H1-elementti on sivun tärkein otsikko — se kertoo sekä käyttäjälle että hakukoneille, mistä sivulla on kyse. Se on hierarkian korkein otsikkotaso ja sitä tulisi käyttää vain kerran per sivu.
<body> <style> h1 { font-size: 32px; text-align: center; border-bottom: 3px solid #444; padding-bottom: 10px; } </style> <h1>Tämä on tyylitelty otsikko</h1> </body>
Tässä esimerkissä muokataan otsikon kokoa ja lisätään alaviiva‑korostus.
